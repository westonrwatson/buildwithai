Foundation

What is AI & Agents?

Foundation

Artificial intelligence

  • Understands language and finds patterns in data
  • Solves problems and creates content
  • Learns from examples — not conscious or alive
The model

Large language models

  • A type of AI trained on huge amounts of text
  • Predicts the next word → essays, code, conversation
  • Claude, ChatGPT, and Gemini are all LLMs
Agents

AI agents

  • Uses an LLM as its brain, then takes action
  • Reads files, writes code, and uses tools over many steps
  • Cursor is an AI agent

In Lab 01 you'll prompt Claude to build your game. In Lab 02 you'll use Cursor — an AI agent — to build a full site with a framework.

Talk to the AI

How to prompt.

Part 01
Role
Tell the AI who it is. Give it a persona that fits the task.
"You are an expert web developer."
"You are a patient coding tutor who explains things simply."
Part 02
Task
Tell it exactly what to build or do. Be direct and specific.
"Build me a Wordle clone."
"Add a dark theme with purple accents to my game."
Part 03
Context
Give it the details, constraints, and requirements it needs.
"Single index.html. No frameworks."
"Vanilla HTML, CSS, and JavaScript only. Must work on mobile."

The build

Wordle clone.

Phase 01
Generate it.
Open claude.ai and paste the starter prompt below. Claude will build a complete, playable Wordle clone as a single index.html file. Download or copy the file when it's ready.
Phase 02
Make it yours.
Pick one customization below and prompt Claude to apply it. Iterate one change at a time until it feels like your version.
Phase 03
Ship it.
Upload your index.html to GitHub, connect to Vercel, and get a live URL. Submit the link below when you're done.
Starter prompt — copy & paste this first

Copy this prompt in English — AI tools work best with English prompts.

You are an expert web developer. Build a fully playable Wordle clone as a single index.html file. Include: 6 guesses per game, 5-letter words only, a built-in word list of common English dictionary words (any standard 5-letter dictionary word should be a valid answer or guess), an on-screen keyboard, tile flip animations, color feedback (green = correct spot, yellow = wrong spot, gray = not in word), win and lose states with a Play Again button, and a clean modern UI. Vanilla HTML, CSS, and JavaScript only — no frameworks, no external libraries. Make it responsive and work on mobile.
Phase 02

Add your flair.

Custom theme
"Switch to a dark theme with [YOUR COLOR] as the accent. Update the keyboard and tiles to match."
Typography
"Change the title and tile letters to fonts that match your style — bold, retro, or minimal."
Tile animations
"Add a satisfying flip animation when each row of tiles is revealed."
Tile style
"Give the tiles rounded corners, a subtle border, and a soft shadow. Match the keyboard keys."
Feedback colors
"Replace the default green, yellow, and gray with your own color palette. Keep the tiles easy to read."
Background
"Add a gradient or patterned background behind the game board. Keep the tiles and text readable."
Phase 03

Push & deploy.

Push to GitHub
  1. Go to github.com and sign in.
  2. Click New (or the + icon) → New repository.
  3. Name it my-wordle and create the repo.
  4. On the empty repo page, click uploading an existing file.
  5. Upload your index.html from Claude, then click Commit changes.
Deploy on Vercel
  1. Go to vercel.com and sign in with GitHub.
  2. Click Add New → Project.
  3. Find my-wordle in your repos and click Import.
  4. Click Deploy — leave the default settings as they are.
  5. When the build finishes, copy your live URL (e.g. my-wordle.vercel.app).
