Core ideas

Design principles.

Judgment calls to keep in mind before you prompt — no tools required, just good taste.

Color

Start with a palette

Use the 60-30-10 rule: 60% neutral background, 30% secondary, 10% accent. Pick one accent color and use it consistently — restraint beats a rainbow.

Type

Two fonts max

One font for headings, one for body. Contrast their weight and size — a bold display face plus a simple body font is enough.

Hierarchy

Guide the eye

The biggest, boldest element should be what matters most. Use size, weight, color, and whitespace — not more colors — to show importance. One clear focal point per section.

Consistency

Repeat what works

Use the same button style, spacing, and corner radius on every section. When the AI adds something new, ask it to match what's already there.

Phase 01

Start from the template.

Download the template and prompt, add your photos, fill in the prompt, and send everything to your AI chat.

Get your files
Download starter file Download prompt See Example
Follow these steps
  1. Download the starter file and prompt — save the starter as index.html
  2. Rename 4 photos as photo1.jpg (hero), photo2.jpg, photo3.jpg, and photo4.jpg — keep them in the same folder as index.html
  3. Open Claude, ChatGPT, or Gemini in your browser
  4. Fill in every field in the prompt document, then copy the prompt section at the bottom
  5. Upload index.html and all 4 photos to the chat, then paste your filled prompt
  6. Preview or download the updated index.html, save it over your file, and open it in your browser
Phase 02

Make it yours.

One change at a time. Paste or upload your current index.html only if the chat no longer has your latest version, then prompt a new change.

Make at least 1 adjustment.

Translate your site
"Translate my entire site into [YOUR LANGUAGE — e.g. Romanian, Russian, Ukrainian]. Keep the same layout and styling. Translate all headings, body text, button labels, and nav links."
Stats highlight
"Restyle my hero stat row — make the numbers bigger and bolder, add subtle dividers between stats, and color the numbers with my accent."
Interest icons
"Swap the icons in my 'Things I care about' list for SVGs from Heroicons that match each interest — [Interest 1], [Interest 2], and [Interest 3]."
Scroll animations
"Add subtle fade-in on scroll for each section and smooth hover effects on buttons, project cards, and value cards."
Values refresh
"Rename the 'More' nav link and section heading to [YOUR TITLE]. Rewrite the four value cards with punchier, more personal copy."
Client brief
"Create a fillable website brief I can send to [CLIENT NAME] before building their About Me site — include questions for their name, tagline, bio, 3 projects, 4 photos, color/style preferences, and social links, plus a prompt template at the bottom they can return to me."
Phase 03

Push & deploy.

Push to GitHub, connect to Vercel, and get a live URL you can share.

Push to GitHub
  1. Go to github.com and sign in.
  2. Click New (or the + icon) → New repository.
  3. Name it about-me and create the repo.
  4. On the empty repo page, click uploading an existing file.
  5. Upload your index.html, then click Commit changes.
Deploy on Vercel
  1. Go to vercel.com and sign in with GitHub.
  2. Click Add New → Project.
  3. Find about-me in your repos and click Import.
  4. Click Deploy — leave the default settings as they are.
  5. When the build finishes, copy your live URL (e.g. about-me.vercel.app).
