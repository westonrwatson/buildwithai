Build Lab 02

Your about me site.

Use Cursor to build a multi-page About Me site with Next.js, then push to GitHub and deploy something you'd put on a resume.

Before you build

Design principles.

Color
Start with a palette
Use the 60-30-10 rule: 60% neutral background, 30% secondary, 10% accent. Generate palettes at Coolors. Check contrast at WebAIM.
Type
Two fonts max
One for headings, one for body. Browse pairings at Google Fonts. Tell the AI: "Use [Font A] for headings and [Font B] for body text."
Components
Don't reinvent buttons
For Next.js projects, use pre-built libraries. Start with shadcn/ui or Material Design 3. Prompt: "Use shadcn/ui for buttons, cards, and navigation."
Space
Whitespace is design
Cramped layouts feel amateur. Ask for "generous padding" and "consistent 8px spacing scale." When in doubt: "Make it feel more spacious and editorial."
Hierarchy
Guide the eye
The biggest, boldest element should be what matters most. Use size, weight, and color — not more colors — to show importance. One clear focal point per section.
Vibe
Describe the feeling
Say "minimal and clean like Apple," "bold and playful like Spotify," or "warm and personal like a letter." The AI understands aesthetic references.
The build

About me site.

Phase 01
Build in Cursor.
Open Cursor, create a new project folder, and paste the starter prompt below. Let the AI scaffold a Next.js site with App Router and Tailwind CSS.
Phase 02
Make it yours.
Build page by page. Use the Design Kit principles — fix one thing at a time. Preview on your phone. Ask the AI to explain anything you don't understand.
Phase 03
Ship it.
Push to GitHub, connect to Vercel, and get a live URL. Follow the Push & Deploy steps below.
Phase 01

Build in Cursor.

One prompt sets up your whole site. Fill in the brackets, paste it once, and let Cursor build.

  1. Open Cursor → Open Folder → create a new folder called about-me
  2. Fill in every [BRACKET] in the starter prompt below
  3. Paste the full prompt in Cursor Chat and let the agent build
  4. Run npm run dev and open localhost:3000 — check every item in the requirements list
Your first version must include
  • 3 pages — Home, About, and Projects (Next.js App Router)
  • Home — your name, a tagline, and a hero section
  • About — a short bio about who you are
  • Projects — at least 3 project or interest cards
  • Navigation — navbar links to every page on every page
  • Contact — your email or a social link in the footer
Starter prompt — fill in the brackets, then paste once

Paste this in Cursor chat — copy in English for best results.

You are an expert Next.js developer. Create a new Next.js project in this folder with App Router, TypeScript, and Tailwind CSS. Build a personal about-me website with exactly these pages: Home (/) — hero with my name [YOUR NAME], tagline "[YOUR TAGLINE]", and a CTA button. About (/about) — short bio [ONE PARAGRAPH ABOUT YOU] plus a list of 3 things I'm passionate about. Projects (/projects) — 3 cards with title, description, and tech tags. Must include: shadcn/ui for buttons, cards, and navigation; shared navbar and footer on every page with links to all pages; footer contact [YOUR EMAIL OR @SOCIAL]; style [e.g. minimal dark theme with lime accents]; font [FONT NAME] from Google Fonts; fully responsive on mobile and desktop. Install all dependencies and make sure npm run dev works.
Phase 02

Make it yours.

One change at a time. Copy a prompt below into Cursor, check the result, then move to the next.

Pick 2 or more adjustments — or write your own in the same style.

Colors & fonts
"Update the site to use [COLOR 1] and [COLOR 2] as my palette. Use [FONT NAME] from Google Fonts for headings and body text."
More breathing room
"Make the layout feel more spacious — increase padding, add more whitespace between sections, and widen the content area on desktop."
Show personality
"Add a hobbies or favorites section on the About page with 3 things that show who I am — music, sports, games, etc."
Smooth animations
"Add subtle fade-in on scroll and smooth hover effects on buttons and project cards."
Dark mode
"Switch the whole site to a dark theme with good contrast. Keep all text easy to read."
Mobile polish
"Improve the mobile layout — stack sections cleanly, fix any overflowing text, and make the nav work well on small screens."
Phase 03

Push & deploy.

Push to GitHub, connect to Vercel, and get a live URL you can share.

Push to GitHub
  1. Go to github.com and sign in.
  2. Click New (or the + icon) → New repository.
  3. Name it about-me and create the repo.
  4. Copy the HTTPS link GitHub shows you for the new repo.
  5. Open Chat in Cursor, paste the link, and ask the agent to push your project to that repo.
Deploy on Vercel
  1. Go to vercel.com and sign in with GitHub.
  2. Click Add New → Project.
  3. Find about-me in your repos and click Import.
  4. Click Deploy — Vercel auto-detects Next.js. Leave the default settings.
  5. When the build finishes, copy your live URL (e.g. about-me.vercel.app).
