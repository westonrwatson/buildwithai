Build Lab 02

Your about me site.

Apply design principles, build a multi-page Next.js site, polish it, and deploy something you'd put on a resume.

Before you build

Design principles.

Color
Start with a palette
Use the 60-30-10 rule: 60% neutral background, 30% secondary, 10% accent. Generate palettes at Coolors. Check contrast at WebAIM.
Type
Two fonts max
One for headings, one for body. Browse pairings at Google Fonts. Tell the AI: "Use [Font A] for headings and [Font B] for body text."
Components
Don't reinvent buttons
For Next.js projects, use pre-built libraries. Start with shadcn/ui or Material Design 3. Prompt: "Use shadcn/ui for buttons, cards, and navigation."
Space
Whitespace is design
Cramped layouts feel amateur. Ask for "generous padding" and "consistent 8px spacing scale." When in doubt: "Make it feel more spacious and editorial."
Hierarchy
Guide the eye
The biggest, boldest element should be what matters most. Use size, weight, and color — not more colors — to show importance. One clear focal point per section.
Vibe
Describe the feeling
Say "minimal and clean like Apple," "bold and playful like Spotify," or "warm and personal like a letter." The AI understands aesthetic references.
The build

About me site.

Stack
Next.js + React.
A multi-page site using the Next.js App Router. This is your capstone — apply the Design Kit and make it something you're proud to share.
Goal
Look professional.
Focus on polish: consistent spacing, a cohesive color palette, smooth navigation, and content that actually represents you.
Outcome
Deploy & share.
Push to GitHub, deploy on Vercel, and submit your live link. This is the site you can put on a resume or send to friends.
Required — your site must have
  • At least 3 pages using Next.js App Router (e.g. Home, About, Projects)
  • Home page with your name, a tagline, and a clear hero section
  • About page with a short bio — who you are, what you're into
  • A projects or interests page with at least 3 items
  • Working navigation that links every page together
  • Contact info — email, social link, or contact form
  • Deployed live on Vercel with a public URL
Make it yours — pick 3+
  • Custom color palette and font pairing from Google Fonts
  • A unique layout — asymmetric, full-bleed images, or card-based
  • A hobbies, music, or favorites section that shows personality
  • Subtle animations — fade-ins, hover effects, page transitions
  • A personal logo, monogram, or custom favicon
  • Dark mode or a theme toggle
  • Responsive design that looks great on phone and desktop
Starter prompt — copy & customize the brackets
You are an expert Next.js developer. Create a multi-page personal about-me website using Next.js 14+ with the App Router and React. Pages: Home, About, and Projects. My name is [YOUR NAME]. I'm [ONE SENTENCE ABOUT YOU]. Style: [DESCRIBE YOUR VIBE — e.g. "minimal dark theme with lime accents"]. Use [FONT NAME] from Google Fonts. Home: hero with name, tagline, and a CTA button. About: bio paragraph and 3 things I'm passionate about. Projects: 3 project cards with title, description, and tech tags. Add a shared navigation bar and footer on every page. Use shadcn/ui for buttons and cards. Make it polished, responsive, and deployable to Vercel.
01 — Scaffold the project

Ask Cursor: "Create a new Next.js project with App Router and Tailwind CSS." Open the folder, run the dev server, and confirm it loads at localhost:3000.

02 — Iterate & polish

Build page by page. Use the Design Kit principles — fix one thing at a time. Preview on your phone. Ask the AI to explain anything you don't understand.

03 — Push to GitHub

Publish to a new repo (e.g. about-me). Make sure .gitignore excludes node_modules.

04 — Deploy on Vercel

Import the repo on Vercel. It auto-detects Next.js. Hit deploy — your site goes live in under a minute. Copy the URL for the showcase below.
